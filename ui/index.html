Plugin Repository by tcnksm

The followings are the experimental cf-plugins by @tcnksm. Everything in this repository is still under hevy development. So please be careful to use. This repository is only for easy way to install & try. After the plugin is getting mature, I will migrate it to official repository.

To use this repository from your cf cli, use `add-plugin-repo` command,

$ cf add-plugin-repo tcnksm http://t-plugin.mybluemix.net
$ cf install-plugin -r tcnksm PLUGIN_NAME
